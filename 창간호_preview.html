주식 시작 가이드 — 계좌 개설부터 실제 매매까지

주식을 한 번도 해본 적 없는 분들을 위해 계좌 개설부터 매도까지 전 과정을 정리했습니다. 순서대로 따라하면 누구든 할 수 있습니다.

1
증권사 선택 — 어디서 사나요? 주식은 은행이 아닌 증권사를 통해 거래합니다. 앱 하나만 있으면 스마트폰으로 언제든 매수·매도 가능합니다.
증권사특징추천 대상
키움증권추천 점유율 1위, 참고 자료 최다 완전 초보
미래에셋 앱 UI 직관적 앱 친숙한 분
삼성증권 삼성 계열 연동 편리 삼성 고객
2
비대면 계좌 개설 — 10분이면 끝 준비물: 신분증 + 본인 명의 휴대폰
① 앱스토어에서 "키움증권 영웅문S" 검색 후 설치
② 앱 실행 → "계좌 개설" 탭 클릭
③ 신분증 촬영 → 얼굴 인식 → 약관 동의
④ 출금 계좌 연결 (평소 쓰는 은행 계좌)
⑤ 완료 → 계좌번호 발급 (무료)
3
증권 계좌에 돈 넣기
① 기존 은행 앱 실행 → "이체"
② 받는 계좌에 키움증권 계좌번호 입력
③ 금액 입력 후 이체 완료
처음엔 10~30만원 소액으로 시작하세요.
잃어도 괜찮은 금액으로 시작해야 냉정하게 배울 수 있습니다.
4
앱에서 종목 검색하기
① 영웅문S 앱 하단 "주식" 탭 클릭
② 검색창에 회사 이름 입력 (예: 삼성전자)
③ 현재가 / 등락률 / 거래량 확인
화면에서 꼭 확인할 것:
현재가 = 지금 이 순간 가격  |  등락률 = 오늘 변동폭(%)  |  거래량 = 오늘 거래된 수량
5
실제 매수 주문 넣기
① 종목 화면에서 "매수" 버튼 클릭
② 주문 유형 선택: 지정가(원하는 가격 직접 입력) 또는 시장가(즉시 체결)
③ 수량 입력 (예: 1주)
④ "매수 확인" 클릭 → 체결 완료
처음엔 지정가로 연습하세요. 가격을 직접 통제하는 감각이 먼저입니다.
6
수익/손실 확인하기 앱 → "잔고" 또는 "보유 주식" 탭에서 실시간 확인

매입가 = 내가 산 가격  |  평가손익 = 현재 수익/손실 금액  |  수익률 = 몇 % 올랐는지

장 시간(오전 9시~오후 3시 30분)에만 가격이 변합니다.
7
매도 — 파는 방법
① 잔고 탭 → 팔고 싶은 종목 클릭
② "매도" 버튼 클릭
③ 수량과 가격 입력 후 확인
④ 수익금은 2영업일 후 출금 가능
💳 수수료 정리
매수 시 약 0.015% · 매도 시 약 0.015% + 거래세 0.18% (모두 자동 차감)
소액 거래에서는 체감이 거의 없는 수준입니다.
